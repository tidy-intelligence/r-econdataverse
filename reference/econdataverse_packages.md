# List all packages in the econdataverse

List all packages in the econdataverse

## Usage

``` r
econdataverse_packages(include_self = TRUE)
```

## Arguments

- include_self:

  Include econdataverse in the list?

## Value

A character vector of package names.

## Examples

``` r
econdataverse_packages()
#>  [1] "cli"           "econdatasets"  "econid"        "econtools"    
#>  [5] "imfapi"        "imfweo"        "oecdoda"       "owidapi"      
#>  [9] "uisapi"        "wbids"         "wbwdi"         "econdataverse"
```
